🧾 Activity Log

ประวัติการแก้ไขข้อมูลลูกบ้าน — อัปเดตโดย resident โดยตรง

กดปุ่ม "🔄 โหลดใหม่" เพื่อโหลด log

📶 การใช้งานบ้าน

สรุปว่าบ้านไหนเข้าใช้งานระบบบ้าง — never / today / last seen / ความถี่ (ไม่ลงรายละเอียดราย transaction)

กดปุ่ม "🔄 โหลดใหม่" เพื่อโหลดข้อมูล

💾 ขนาดข้อมูล (โดยประมาณ)

สรุปคร่าวๆ ว่าแต่ละหน้าที่เก็บรูป/ไฟล์ใช้พื้นที่ไปเท่าไหร่ — ตัวเลขเป็นการประมาณจากข้อมูลที่อ่านได้ ไม่ใช่ค่าที่แน่นอน 100%

กดปุ่ม "🔄 โหลดข้อมูล" เพื่อเริ่มตรวจสอบ (จะอ่านข้อมูลจากหลายหน้า อาจใช้เวลาสักครู่)
📌 ลิมิตของ Firebase ที่ควรรู้ (Spark/free tier)
• Firestore: เก็บข้อมูลได้ฟรีรวม ~1 GiB, เอกสาร (document) แต่ละชิ้นห้ามเกิน 1 MiB — ลิมิตนี้บังคับทุกแผน ไม่ใช่แค่ฟรี (หน้า "แจ้งปัญหา" เก็บรูปเป็น base64 ฝังในเอกสารโดยตรง จึงเสี่ยงชนลิมิตนี้ที่สุด)
• Cloud Storage: พื้นที่ฟรี ~5 GB, ดาวน์โหลดฟรี ~1 GB/วัน — ถ้าเกินจะเริ่มมีค่าใช้จ่ายตามการใช้งานจริง (ไม่ได้ถูกบล็อกทันที ถ้าผูก billing account/Blaze plan ไว้)
• ตัวเลขข้างบนเป็นการประมาณจากหน้านี้เท่านั้น หากต้องการตัวเลขที่แน่นอน 100% ของทั้งโปรเจกต์ ให้ดูที่ Firebase Console → Usage/Storage ของแต่ละโปรเจกต์ (pattra8-54c3f และ pattra8-expense)
